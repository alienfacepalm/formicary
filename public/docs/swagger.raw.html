Formicary API

The formicary is a distributed orchestration engine based on Leader-Follower, Pipes-Filter, Fork-Join and SEDA design principles for executing a directed acyclic graph of tasks, which is also referred as a job workflow. A task represents a unit of work and a job definition is used to specify the task dependencies in the graph/workflow including configuration parameters and conditional logic.
